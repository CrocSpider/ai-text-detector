# Implementation Roadmap

## Phase 1

- ship the current scaffold
- validate extraction and result schema
- connect frontend and backend in local and containerized environments

## Phase 2

- replace the classifier placeholder with a fine-tuned encoder
- add stylometric model training and a learned meta-ensemble
- add benchmark runner and threshold explorer

## Phase 3

- add Redis queue, persistent storage, and retention controls
- add model versioning, drift monitoring, and admin comparison views
- add multilingual support behind explicit capability flags
