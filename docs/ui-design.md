# UI Design

## Primary surfaces

- Home analyzer page with one upload panel, one text area, and one primary action.
- Result card with score, confidence, recommendation, and expandable explanations.
- Result details page for single-document review.
- Hidden admin evaluation page for benchmark snapshots.

## Design tokens

- Background: `#F8F8F6`
- Surface: `#FFFFFF`
- Border: `#E7E5E4`
- Text: `#111827`
- Low risk: `#166534`
- Moderate risk: `#B45309`
- Elevated risk: `#B91C1C`
- Inconclusive support text: `#6B7280`

## Accessibility

- keyboard-usable upload controls
- visible focus states
- plain-language warning text
- non-color cues through labels and score badges
