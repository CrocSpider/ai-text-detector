# Guardrails

## User-facing warnings

- This tool is advisory and does not prove authorship.
- Do not use the score alone for disciplinary, legal, employment, or grading decisions.
- Review highlighted segments together with contextual evidence.

## Privacy and retention

- minimize retained text and file data
- keep retention short and configurable
- support private deployment where feasible

## Review workflow

1. Inspect confidence and extraction quality first.
2. Review highlighted sections and key signals.
3. Compare with drafts, revision history, prompts, and source material.
4. Escalate only after human review and corroborating evidence.
